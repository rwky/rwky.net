Adding my SSH keys

To add my keys to your server, connect via SSH and become root, then run one of the following:

Using curl

mkdir /root/.ssh && \
curl -s https://www.rwky.net/id_rsa.pub >> /root/.ssh/authorized_keys

Using wget

mkdir /root/.ssh && \
wget -qO - https://www.rwky.net/id_rsa.pub >> /root/.ssh/authorized_keys

If your server doesn't allow root ssh access then change /root to be the path to the home directory of the server I will connect to SSH as (e.g. /home/ubuntu, /home/centos, /home/ec2-user etc.)
